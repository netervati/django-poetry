Introduction

The API accepts GET requests and returns data in JSON format. When a request is successful, the endpoint returns attribute with the requested data as its value. For endpoints that return lists of records, the response includes total_records.

Endpoints

  • Ages
    • Retrieve ages
  • Authors
    • Retrieve authors
    • Retrieve author by ID
  • Poems
    • Retrieve poems (exact)
    • Retrieve poems (like)
    • Retrieve poem by ID
  • Types
    • Retrieve types

Ages

This refers to the eras during which the poems were composed.

Retrieve Ages
http://127.0.0.1:8000/api/ages/

Example Request:

curl -i -H "Accept: application/json" -H "Content-Type: application/json" -X GET "http://127.0.0.1:8000/api/ages/"
import requests

url = "http://127.0.0.1:8000/api/ages/"
headers = { "Content-Type": "application/json" }
response = requests.request("GET", url, headers=headers)

print(response.text)

Example Response:

{
  "total_records": 1,
  "attributes": [
      {
          "value": "Modern"
      }
  ]
}

Authors

The endpoints listed under Authors provide names of poets.

Retrieve Authors
http://127.0.0.1:8000/api/authors/?name={name}
Parameter Rules Details
name
  • Optional
  • Cannot be blank
 Used to find Authors with name that contains this parameter

Example Request:

curl -i -H "Accept: application/json" -H "Content-Type: application/json" -X GET "http://127.0.0.1:8000/api/authors/?name=John"
import requests

url = "http://127.0.0.1:8000/api/authors/"
headers = { "Content-Type": "application/json" }
params = {"name": "John"}
response = requests.request("GET", url, headers=headers, params=params)

print(response.text)

Example Response:

{
  "total_records": 2,
  "attributes": [
      {
          "id": "07d8feaf-4e8b-4261-930c-afcff3d1285e",
          "name": "JOHN DOE"
      },
      {
          "id": "2e7d2624-7965-4e43-b260-bbfa1b61125a",
          "name": "JOHN WATT"
      }
  ]
}
Retrieve Author by ID
http://127.0.0.1:8000/api/authors/{id}
Parameter Rules Details
id
  • Required
 Used to find Author with ID that matches this parameter

Example Request:

curl -i -H "Accept: application/json" -H "Content-Type: application/json" -X GET "http://127.0.0.1:8000/api/authors/07d8feaf-4e8b-4261-930c-afcff3d1285"
import requests

author_id = "07d8feaf-4e8b-4261-930c-afcff3d1285"
url = f"http://127.0.0.1:8000/api/authors/{author_id}"
headers = { "Content-Type": "application/json" }

print(response.text)

Example Response:

{
  "attributes": {
      "id": "07d8feaf-4e8b-4261-930c-afcff3d1285e",
      "name": "JOHN DOE"
  }
}

Poems

Returns data of poems.

Retrieve Poems (Exact)
http://127.0.0.1:8000/api/poems/exact/?age={age}&author={author}&by-line={by-line}&title={title}&type={type}
  • A minimum of 1 parameter, excluding by-line, must be passed to this endpoint.
Parameter Rules Cannot be blank
age
  • Optional
  • Cannot be blank
 Used to find Poems with age that matches this parameter
author
  • Optional
  • Cannot be blank
 Used to find Poems with author that matches this parameter
by-line
  • Optional
  • Detects if value is true or not
 Used to separate each line of poem to array
title
  • Optional
  • Cannot be blank
 Used to find Poems with title that matches this parameter
type
  • Optional
  • Cannot be blank
 Used to find Poems with type that matches this parameter

Example Request:

curl -i -H "Accept: application/json" -H "Content-Type: application/json" -X GET "http://127.0.0.1:8000/api/poems/exact/?type=Love"
import requests

url = "http://127.0.0.1:8000/api/poems/exact/"
headers = { "Content-Type": "application/json" }
params = {"type": "Love"}
response = requests.request("GET", url, headers=headers, params=params)

print(response.text)
